# IT & Cybersecurity Home Lab

This repository documents my hands-on journey into IT support and cybersecurity through a practical home lab environment. I am using virtual machines, real tools, and structured exercises to build a strong foundation in Windows administration, Linux, networking, troubleshooting, and security.

My current focus is on developing real-world technical skills that align with entry-level IT support roles while also building toward a future transition into cybersecurity.

## Purpose

The goal of this home lab is to move beyond theory and gain experience through direct practice. Rather than only studying concepts, I am setting up systems, installing tools, troubleshooting issues, documenting what I learn, and building familiarity with the kinds of tasks that show up in real IT and security environments.

This repo serves as both:
- a personal learning journal
- a record of hands-on technical progress
- a growing portfolio of lab work and notes

## Current Direction

My long-term goal is cybersecurity, but right now I am focused on building a stronger foundation in IT, operating systems, troubleshooting, and computer fundamentals. I see this as an important step toward becoming more effective in security later on.
